# 🧬 Diagnogenie - Disease Prediction Web App 🌐

Diagnogenie is a web-based application that predicts diseases based on user-selected symptoms. Built using HTML, CSS, JavaScript, and a backend database (MySQL), the project demonstrates a real-life implementation of a medical diagnosis system.

---

## 🚀 Features

- 🧾 User can select multiple symptoms
- 🤖 System predicts likely diseases
- 👩‍⚕️ Admin panel to manage diseases, symptoms, and feedback
- 📊 Stores user feedback and interaction history
- 🎯 Responsive UI with clean navigation
- 🗃️ Database-backed logic using MySQL / PHPMyAdmin

---

## 🛠️ Tech Stack

| Layer     | Technology Used          |
|-----------|---------------------------|
| Frontend  | HTML, CSS, JavaScript     |
| Backend   | PHP / XAMPP / MySQL       |
| Database  | MySQL (PHPMyAdmin)        |
| Hosting   | Localhost (XAMPP Server)  |
